# SewerGEMS Learning Canvas App

## Setup

### 1 Install dependencies
pip install -r requirements.txt

### 2 Add API key
Edit fetch_videos.py

### 3 Fetch videos
python fetch_videos.py

### 4 Run app
streamlit run app.py

---

## Features

- Canvas topic UI
- Exact video playback
- Playlist mode
- Notes system
- Progress tracking
- Bookmark topics
